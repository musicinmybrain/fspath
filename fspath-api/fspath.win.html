fspath.win module¶

Windows stuff

fspath.win.wrapScriptExe(script, shebang='#!python.exe')[source]¶

Wraps a single script into a MS-Win .exe.

Only the script file contents is wraped into the .exe, not the whole python environment!

This is usefull to create .exe console scripts for python entry points, which can be called directly (myscript.exe instead python myscript.py).

Caution

  • This is in an experimental state!
  • This makes use of undocumented pip APIs (ATM pip has no offical API)
  • Use it with care!
  • Shebang is always #!python.exe
